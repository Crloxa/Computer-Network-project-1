# VisualNet-QR-Video 项目快速配置指南

## 项目概述

这是一个基于仿QR码的视频数据传输系统，可以将任意文件编码为QR码序列并生成视频，也可以从视频中解码还原原始文件。

## 快速开始

### 第一步：下载项目代码
```bash
git clone https://github.com/XuYifan-423/VisualNet-QR-Video.git
cd VisualNet-QR-Video
```

### 第二步：下载依赖库（单独下载）

由于依赖库文件较大，请从以下链接下载：

#### OpenCV 4.1.2（必需）
- **下载地址**：https://opencv.org/releases/ 或 https://github.com/opencv/opencv/releases/tag/4.1.2
- **文件**：`opencv-4.1.2-vc14_vc15.exe`

#### FFmpeg（必需）
- **下载地址**：https://ffmpeg.org/download.html 或 https://www.gyan.dev/ffmpeg/builds/
- **文件**：`ffmpeg-release-full.7z`

### 第三步：配置依赖库目录

在项目根目录创建以下结构：

```
Viusal_Net/
├── opencv/          # 解压OpenCV到这里
│   └── build/
├── ffmpeg/          # 解压FFmpeg到这里
│   └── bin/
└── newvisualnet/    # 源代码（已包含）
```

**具体操作：**
1. 运行OpenCV安装程序，选择解压到 `opencv` 目录
2. 解压FFmpeg到 `ffmpeg` 目录
3. 确保最终目录结构正确

### 第四步：打开项目

1. 用Visual Studio打开 `newvisualnet.sln`
2. 选择 `x64` 平台和 `Debug` 配置
3. 编译项目（Ctrl+Shift+B）

### 第五步：运行程序

编译成功后，在 `x64/Debug/` 目录下会生成 `newvisualnet.exe`

**基本用法：**
```bash
# 编码文件为QR码视频
newvisualnet.exe qr-encode input.jpg

# 解码QR码视频还原文件
newvisualnet.exe qr-decode qr_video.mp4

# 查看帮助
newvisualnet.exe --help
```

## 项目文件说明

### 核心源代码文件：
- `main.cpp` - 主程序入口
- `Encoder.cpp/h` - 编码器实现
- `Decoder.cpp/h` - 解码器实现
- `QRPosition.cpp/h` - QR码布局管理
- `FFmpegWrapper.cpp/h` - FFmpeg封装
- `Config.h` - 配置参数

### 配置文件：
- `newvisualnet.vcxproj` - Visual Studio项目文件
- `newvisualnet.sln` - 解决方案文件

### 输出目录：
- `x64/Debug/` - 编译输出和运行时文件
- `x64/Debug/input/` - 输入文件目录
- `x64/Debug/output/` - 输出文件目录

## 常见问题

### Q: 编译时出现"找不到opencv2/opencv.hpp"错误
A: 检查OpenCV是否正确解压到 `opencv/build/include/` 目录

### Q: 链接时出现"找不到opencv_world4120.lib"错误
A: 检查OpenCV库文件是否在 `opencv/build/x64/vc16/lib/` 目录

### Q: 运行时出现"缺少DLL"错误
A: 将OpenCV的DLL文件从 `opencv/build/bin/` 复制到 `x64/Debug/` 目录

### Q: FFmpeg相关错误
A: 检查 `ffmpeg/bin/ffmpeg.exe` 文件是否存在
